# Portfolio
This is a single page application web portfolio that will feature all my projects and details about my education, work experience and skills


Technologies used: 

1. Node js
2. Express server
3. SourceTree, GIT GUI
4. Bootstrap
5. JQuery
6. Nodemon
7. Fontawesome
8. Iconify

Installation:

1. clone the repo
2. run npm install
3. run npm start
